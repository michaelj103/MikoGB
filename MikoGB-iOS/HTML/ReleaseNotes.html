Release Notes

----- Version 0.10.x -----
Game Boy Color Support!

Version 0.10.7

  • Updated networking package dependencies

Version 0.10.6

  • Added an app icon at last
  • Added release notes. Hello, reader!

Version 0.10.5

  • Fixed a couple of issues that could lead to the real-time clock drifting behind gradually

Version 0.10.4

  • Re-signed app for certificate expiration

Version 0.10.3

  • Fixed an issue with MBC5 banking

Version 0.10.2

  • Support for MBC5 games
  • Support for Game Boy Color double-speed CPU mode, required by most MBC5 games
  • Support for MBC5 + Rumble games (e.g. Pokemon Pinball). Rumble output is ignored...for now

Version 0.10.1

  • Support for MBC3 + Real Time Clock (RTC)
  • Defined RTC save data format
  • RTC data is pegged to system time whenever the cartridge writes to its clock and saved alongside save data

Version 0.10

  • Pretty big rendering engine refactor
  • Switchable WRAM and VRAM banks
  • New DMA transfer process for color sprites

----- Version 0.9.x -----
Link Session Support!

Version 0.9.1

  • Fix an issue with making initial connection
  • Disable fast emulation when a link connection is active
  • (Server-side) require client API version to participate in link sessions
  • (Client-side) report client API version. Clients below v0.9.1 will be rejected

Version 0.9

  • Emulator Core API for emitting and receiving serial cable messages
  • Adoption of shared client-side connection framework for communicating with link cable servers
  • Infrastructure for managing the link cable session with the HTTP coordination server
  • Link management UI visible from within a game session
  • (Debug) settings to target a dev server instance
